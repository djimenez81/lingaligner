# `lingaligner`

**WARNING:** At the present moment this package is at the very early stages of
implementation, therefore it is not yet functional.

The `lingaligner` package is a small library that provides classes and
functionalities to the user to perform the alignment of cognates or
corresponding terms within a user-provided corpora according to a given set of
criteria, including defined distances (as feature-base distance) in an
analogous way as sequence are align in genetics or proteomics.
